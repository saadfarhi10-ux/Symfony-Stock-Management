# Symfony Stock Management Application

This project is a web application developed with PHP and Symfony that allows the management of products and stock in a store.

## Features

- User authentication
- Product management
- Category management
- Stock movement tracking
- CRUD operations

## Technologies

- PHP
- Symfony
- MySQL
- Twig
- Doctrine ORM

## Installation

1. Clone the repository
2. Run `composer install`
3. Configure the `.env` database connection
4. Run `php bin/console doctrine:migrations:migrate`
5. Start the Symfony server
